# Contributing

Thanks for your interest.

## How to contribute

1. Fork this repo and create a feature branch.
2. Update skill files (`SKILL.md`, `references/*`, `agents/openai.yaml`) with clear intent.
3. Keep safety boundaries and risk rules consistent.
4. Open a PR with a short summary and sample prompts.

## Guidelines

- Prefer evidence-backed updates in `references/research/`.
- Mark uncertain lines as community-spread phrases.
- Avoid adding harassment, hate, or discriminatory language.
